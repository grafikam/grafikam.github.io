How to use logarithm. Press “log(“ (or L on keyboard) button. In the first bracket write base of the log ,then close the bracket and press “_” button (shift + “-“ on keyboard).In the opened bracket write down logarithm argument and close the bracket.
Some not obvious shortcuts.
R – square root
S – sinus
C – cosinus
Enter - run function
1) If -X is not working try 0-X.
2) You can generally skip “ * ” in obvious cases (e.g. 5x) but if it does not work try to put it.
3) Avoid using meaningless brackets.
